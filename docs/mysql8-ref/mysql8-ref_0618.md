10.11.5 外部锁定

MySQL 使用锁定来管理对表内容的争用：

+   内部锁定是在 MySQL 服务器内部执行的，通过多个线程管理对表内容的争用。这种类型的锁定是内部的，因为它完全由服务器执行，不涉及其他程序。参见第 10.11.1 节，“内部锁定方法”。

+   外部锁定发生在服务器和其他程序锁定`MyISAM`表文件以协调它们之间哪个程序可以在哪个时间访问表。参见第 10.11.5 节，“外部锁定”。
