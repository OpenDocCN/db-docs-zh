MySQL 支持资源组的创建和管理，并允许将在服务器内运行的线程分配到特定的组，以便线程根据组可用的资源执行。本节描述了用于资源组管理的 SQL 语句。有关资源组功能的一般讨论，请参见第 7.1.16 节，“资源组”。
