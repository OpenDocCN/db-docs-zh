#### 15.7.7.20 显示函数状态语句

```sql
SHOW FUNCTION STATUS
    [LIKE '*pattern*' | WHERE *expr*]
```

这个语句类似于`SHOW PROCEDURE STATUS`，但用于存储函数。参见 Section 15.7.7.28, “SHOW PROCEDURE STATUS Statement”。
