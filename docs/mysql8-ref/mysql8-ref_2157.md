#### 30.4.3.20 memory_global_total 和 x$memory_global_total 视图

这些视图总结了服务器内的总内存使用情况。

`memory_global_total` 和 `x$memory_global_total` 视图具有以下列：

+   `total_allocated`

    服务器内分配的总字节内存。
