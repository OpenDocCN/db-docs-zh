28.8 SHOW 语句的扩展

`INFORMATION_SCHEMA` 提供对数据库元数据的访问，以及关于 MySQL 服务器的信息，如数据库或表的名称、列的数据类型或访问权限。有时用于表示这些信息的其他术语包括数据字典和系统目录。
