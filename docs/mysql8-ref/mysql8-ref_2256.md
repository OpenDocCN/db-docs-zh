MySQL 企业版是一款商业产品。与 MySQL 社区版一样，MySQL 企业版包括 MySQL 服务器，一个完全集成的事务安全、ACID 兼容的数据库，具有完整的提交、回滚、崩溃恢复和行级锁定功能。此外，MySQL 企业版还包括以下组件，旨在提供监控和在线备份，以及改进的安全性和可扩展性：
