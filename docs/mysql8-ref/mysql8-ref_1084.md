#### 15.7.7.8 SHOW CREATE FUNCTION Statement

```sql
SHOW CREATE FUNCTION *func_name*
```

这个语句类似于`SHOW CREATE PROCEDURE`，但用于存储函数。请参阅第 15.7.7.9 节，“SHOW CREATE PROCEDURE Statement”。
