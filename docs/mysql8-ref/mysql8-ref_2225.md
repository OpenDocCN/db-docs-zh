#### 30.4.5.8 `list_drop()` 函数

从逗号分隔的值列表中删除一个值并返回结果。更多信息，请参阅 `list_add()`` 函数") 的描述。

##### 参数

+   `in_list TEXT`: 要修改的列表。

+   `in_drop_value TEXT`: 要从列表中删除的值。

##### 返回值

一个 `TEXT` 值。
