#### 13.4.2.11 MultiLineString 类

一个`MultiLineString`是由`LineString`元素组成的`MultiCurve`几何集合。

**`MultiLineString` 示例**

+   在区域地图上，一个`MultiLineString`可以代表一个河流系统或者高速公路系统。
