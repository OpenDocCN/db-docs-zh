#### 19.5.1.5 复制`CREATE SERVER`、`ALTER SERVER`和`DROP SERVER`

`CREATE SERVER`、`ALTER SERVER`和`DROP SERVER`语句不管使用何种二进制日志格式，都不会被写入二进制日志中。
